Dataset Description

This dataset contains about 60,000 Instagram users. The dataset contains about 2,600,000 posts from these users. Each post contains a text description and a image. Image files are in JPG format. The metadata file contains the number of fans, the number of followers and other data characteristics of the user, as well as the posting text and posting picture id of the user.
